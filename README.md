# Income_and_Expense_Tracker
# 💰 Smart Expense Tracker Dashboard
## 📌 Project Overview
A mobile-friendly web application to track daily expenses, manage income, and visualize financial data using an interactive dashboard.

This project focuses on usability, real-time insights, and a clean UI design, making it more like a real product than a basic student project.
---
## 🚀 Features
### 📊 Dashboard
- Total Balance, Income, Expense cards
- Pie chart for category-wise spending
- Line chart for expense trends
- Subcategory analytics
- Smart insights (highest spending category)
### 🧾 Transactions
- Add, Edit, Delete transactions
- Category & Subcategory support
- Income / Expense tracking
- Automatic date handling
### 🔍 Search & Filters
- Search by category
- Filter by date
- Filter by date range
### 📅 Views
- Dashboard view
- Today’s expenses
- All transactions (table)
- Calendar view (grouped by date)
### 🔐 Role-Based UI
- Admin: Full access (Add/Edit/Delete)
- Viewer: Read-only access
### 📥 Export & Backup
- Download by date, month, category
- Backup data (JSON)
- Restore data from file
### 🌐 UI Features
- Dark mode 🌙
- Multilingual support (English, Telugu, Hindi)
- Responsive design (mobile-friendly)
- Modern card-based layout
---
## 🛠️ Tech Stack
- React.js
- Recharts (for charts)
- CSS (custom styling)
- LocalStorage (for data persistence)
---
## 💾 Data Storage
- Uses browser localStorage
- Works offline
- Data persists unless browser storage is cleared
---
## 🧠 Key Highlights
- One-click expense tracking UX
- Subcategory-based analytics
- Calendar-based expense view
- Role-based interface simulation
- Backup & restore without backend
---
## ⚠️ Limitations
- No backend/database
- Data is device-specific
- No authentication system
